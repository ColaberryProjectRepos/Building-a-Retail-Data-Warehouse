Steps to Create Data Bases for the Project As Required 9

In other to complete the Data Base creation, you will have to follow the below listed

  • Go to SQL Server Management Studio and create these three Data Bases.
  •  Staging_Data Base
  • Project1_ODS
  • Project_1_Data-warehouse



To create the Data Bases,  follow the these steps below.

  • Open SQL Server Management Studio
  • Right Click On Data Base
  • Select New
  • Enter The Data Base Name
  • Press Ok
  • Refresh your Data Base Icon for your New Data Base to appear
