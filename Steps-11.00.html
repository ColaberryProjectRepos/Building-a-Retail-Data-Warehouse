Loading Dimension Tables 11

Loading Dimension tables are unique processes that is done in SSIS.

The below listed steps are necessary in loading data into dimension tables

  • Create SSIS package to load these table.
  • Click on Data flow tab to open the Canvas
  • Drag and drop the Ole DB source tool to the canvas and configure
  • Drag and drop the Lookup tool onto the canvas and configure and connect to Ole DB source
  • Drag and drop derived column tool to create new needed columns
  • Drag and drop the Ole DB Destination Column to the canvas and configure
  • Execute your package for data to load into your tables
